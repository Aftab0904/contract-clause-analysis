# Contract AI Engine

AI-powered system to analyze legal contracts using RAG.

## Features
- Clause extraction (10 categories)
- Risk detection
- Natural language QA
- Source tracking

## Tech Stack
- Python
- Streamlit
- LangChain
- ChromaDB
- Groq LLM
